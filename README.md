# zavrsniRad

Tema: Baza filmova 
Aplikacija ce imati 3 strane:

Pocetna (stranica sa opstim podaima)
Pretraga(bice vrsena po nazivu filma a filmovi ce biti sortirani po razlicitim kriterijumima)
Dodaj film(forma sa input poljima)
Za izradu aplikacije ce biti koriscen html,css,bootstrap,javascript i jqury.
